# 기술 활용 로드맵

오늘 수업은 Chat UI에서 시작해 Build를 거쳐 CLI Agent까지 올라갑니다. NotebookLM 스튜디오 슬라이드 `Enterprise AI Roadmap (2)`를 내려받아, 각 단계의 역할과 오늘 사용할 도구를 실제 deck 기준으로 연결합니다.

## Chat UI

- Gemini와 NotebookLM으로 질문, 리서치, 보고서 초안, 수정 요청을 반복하는 협업 루프
- 사람이 목표와 맥락을 잡고 AI가 반복 속도를 높이는 단계
- 이동: `#ch03-clip01`

## API / Build

- AI Studio Build와 Gemini API로 앱을 더 빨리 만들고, 완성된 앱이 다시 AI를 호출하는 구조
- 맞춤형 서비스와 업무 자동화를 구현하는 단계
- 이동: `#ch06-clip01`

## CLI / Agent

- 터미널 기반으로 계획, 도구 호출, 실행, 검증, 수정을 반복하는 agentic workflow
- 오늘은 Codex와 Cline으로 시연
- 이동: `#ch07-clip01`

## 오늘의 여정

오전에는 Chat UI로 Gemini와 NotebookLM을 체험하고, 이후 API / Build로 AI Studio Build와 Gemini API를 다룬 뒤, 마지막에는 Codex와 Cline으로 agentic workflow를 시연합니다.
