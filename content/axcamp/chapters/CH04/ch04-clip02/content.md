# 문서 기반 AI 리서치: CIQO와 LG 스타일 브리핑

NotebookLM의 핵심은 **좋은 소스를 넣고, 출처가 붙은 답을 뽑아내고, 그 결과를 다시 산출물로 전환하는 것**입니다.
이번 실습에서는 WEF와 Deloitte 보고서, Fast Research, 노트 전환, Studio 패널, 그리고 CH02에서 만든 JSON 슬라이드 템플릿까지 연결해 **LG 스타일 브리핑**으로 완성합니다.

## 2. 소스 업로드 안내

### 소요 시간 약 3분

NotebookLM 노트북을 생성하고 분석 대상 소스를 업로드하는 단계입니다.

- [NotebookLM (notebooklm.google.com)](https://notebooklm.google.com/)에 접속합니다.

- **새 노트북**을 클릭합니다.

- 노트북 이름을 **AX Camp 전략 리서치**로 설정합니다.

- **소스 추가** → 파일 업로드를 선택합니다.

- 위 4개 파일을 순서대로 업로드합니다.

### CIQO 포인트

NotebookLM은 소스 품질과 조합에 따라 답변 수준이 달라집니다. WEF는 거시 데이터, Deloitte는 산업 맥락, LG 로고는 브랜딩 힌트 역할을 합니다.

## 3. 기본 질문 프롬프트: 다중 소스 비교 분석

### 소요 시간 약 5분

여러 소스를 교차 분석하는 NotebookLM의 핵심 가치를 시연하는 구간입니다. 프롬프트 3개를 순차 진행하세요.

프롬프트 2-1: 단일 소스 심층 질의복사

WEF 미래 직업 보고서에서 AI로 인해 가장 크게 성장할 것으로 전망되는
직무 TOP 5와, 가장 크게 감소할 직무 TOP 5를 정리해 줘.

각 직무별로:
- 2025-2030 예상 변화율 (%)
- 변화의 주요 원인
- 해당 직무가 많은 산업군
을 표로 만들어 줘.

프롬프트 2-2: 다중 소스 교차 분석복사

WEF 보고서의 '기술 변화가 산업에 미치는 영향' 분석과
딜로이트 럭셔리 보고서의 '디지털 전환 트렌드'를 교차 비교해 줘.

두 보고서에서 공통으로 강조하는 트렌드는 무엇이고,
서로 다른 시각을 보이는 부분은 무엇인지 정리해 줘.

프롬프트 2-3: 비즈니스 임팩트 분석복사

두 보고서의 내용을 종합했을 때,
LG그룹 같은 대기업 임원이 주목해야 할 시사점 3가지를 뽑아줘.

각 시사점마다:
- 근거 데이터 (어느 보고서 몇 페이지)
- 임원 액션 아이템
- 리스크와 기회
를 구분해서 정리해 줘.

## 4. 패스트 리서치 프롬프트

### 소요 시간 약 4분

NotebookLM의 Fast Research 기능으로 웹 소스를 자동 수집하는 흐름입니다.

- 좌측 소스 패널에서 **소스 추가**를 클릭합니다.

- **패스트 리서치(Fast Research)** 또는 **웹 검색**을 선택합니다.

- 아래 주제를 입력합니다.

- NotebookLM이 관련 웹 소스를 자동으로 수집해 노트북에 추가하는지 확인합니다.

패스트 리서치 주제복사

2025-2026년 글로벌 AI 가전 시장 트렌드와 주요 기업(LG, 삼성, 하이센스)의 AI 전략 비교 분석

대안 주제복사

한국 대기업의 AI 인재 양성 전략과 리스킬링 프로그램 최신 동향

## 5. 노트를 소스로 전환

- 좌측 노트 패널에서 저장된 노트를 선택합니다.

- 노트 상단 또는 우클릭 메뉴에서 **소스로 전환**을 클릭합니다.

- 해당 노트가 소스 목록에 추가된 것을 확인합니다.

전환 후 활용 프롬프트복사

방금 소스로 전환된 '임원 시사점 분석' 노트와 원래 WEF/딜로이트 보고서를 함께 참조해서, 각 시사점별 실행 계획을 구체화해 줘. 특히 LG그룹 임원이 다음 분기에 바로 착수할 수 있는 액션 아이템 중심으로 정리해 줘.

## 6. 스튜디오 패널 활용 프롬프트

### 소요 시간 약 4분

NotebookLM 스튜디오에서 슬라이드와 오디오 오버뷰를 생성해 보는 단계입니다.

- 우측 **스튜디오** 패널을 엽니다.

- 생성할 콘텐츠 유형을 선택합니다.

6-A: 슬라이드 생성 프롬프트복사

AI 시대 임원 리더십 전략 - 핵심 데이터 요약

대상 청중: LG그룹 C-Level 임원
슬라이드 수: 8~10장
스타일: 간결하고 데이터 중심, 경영진 보고 형식
포함 내용:
- WEF 보고서의 핵심 수치
- 딜로이트 보고서의 시장 트렌드
- 임원 액션 아이템 요약

6-B: 팟캐스트(오디오 오버뷰) 생성 프롬프트복사

두 명의 전문가가 AI 시대의 기업 전략에 대해 대화하는 형식으로 만들어 줘.
특히 LG그룹 같은 한국 대기업의 관점에서 논의해 줘.
핵심 데이터를 자연스럽게 대화에 녹여서 설명해 줘.

## 7. JSON 활용 슬라이드 작성

CH02에서 수행한 **JSON 활용 슬라이드 템플릿**을 여기서 다시 활용합니다. NotebookLM에서 정리한 핵심 내용과 JSON 템플릿을 함께 넣으면,
슬라이드 구조를 유지한 채 LG 스타일 브리핑 초안을 훨씬 빠르게 만들 수 있습니다.

JSON 템플릿 기반 슬라이드 작성 요청복사

아래 두 자료를 바탕으로 LG그룹 임원 보고용 슬라이드 초안을 작성해 줘.

자료 1: CH02에서 만든 JSON 슬라이드 템플릿
자료 2: NotebookLM이 정리한 WEF/딜로이트/패스트 리서치 기반 요약 내용

작성 조건:
- JSON 템플릿의 장표 구조를 유지할 것
- 각 장표는 제목, 핵심 메시지, 근거 데이터, 시사점을 포함할 것
- 톤은 간결하고 데이터 중심의 LG 스타일 브리핑으로 맞출 것
- 마지막 장에는 임원 액션 아이템 3개를 넣을 것
