참고
유료 API KEY 발급
GCP 무료 크레딧과 연동하여 유료 Gemini API 키를 발급받는 과정입니다. 사전 준비가 완료되지 않은 경우 아래 영상을 참고하세요.
영상 가이드

GCP 가입부터 API 키 발급까지 전체 과정 영상

Step 1. API 키 메뉴 선택

화면 좌측 메뉴 중 열쇠 모양 아이콘, 혹은 [Get API key] 버튼을 클릭합니다.

좌측 사이드바 메뉴의 'Get API key' 클릭

Step 2. API 키 생성 및 프로젝트 연결

[API 키 만들기] 버튼을 누른 뒤, 반드시 기존 프로젝트 연결을 선택해야 합니다.

옵션 선택: [프로젝트 가져오기]
프로젝트 선택: My First Project (GCP 가입 시 생성된 프로젝트)
주의
'새 프로젝트에 API 키 만들기'를 누르면 무료 크레딧이 연동되지 않을 수 있습니다. 반드시 [프로젝트 가져오기]를 선택하세요.

'프로젝트 가져오기' 버튼 클릭

드롭다운에서 My First Project 선택

Step 3. 키 복사 및 보관

잠시 로딩 후 생성된 API 키가 목록에 나타납니다.

생성된 키 옆의 [Copy] 버튼을 눌러 메모장 등에 복사해 둡니다
실습 당일 이 키가 없으면 진행이 불가능하니 꼭 저장해 주세요

API 키 생성 완료 (리스트 및 복사 아이콘)

API Key 보안 주의
API Key는 비밀번호와 같습니다. 이메일, 메신저, 공개 문서에 절대 공유하지 마세요. 키가 유출되면 타인이 본인의 크레딧을 사용할 수 있습니다.
