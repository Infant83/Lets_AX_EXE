ChatGPT 및 GPTs 소개

ChatGPT는 일반 채팅만 하는 도구가 아니라, Deep Research와 GPTs를 통해 반복 업무를 구조화할 수 있는 작업 플랫폼입니다. 이번 클립에서는 GPTs를 짧게 소개하고, AI 인프라 수혜주 심층 리서치 결과를 PPT로 정리하는 실습 흐름을 단계별로 확인합니다.
1. ChatGPT에서 먼저 볼 것
  1. chatgpt.com에 접속해 로그인합니다.
  2. 일반 채팅, Deep Research, GPTs처럼 작업 유형이 어떻게 나뉘는지 먼저 확인합니다.
  3. 오늘은 단순한 질문보다, 반복 가능한 조사와 문서화 흐름을 만드는 관점으로 ChatGPT를 봅니다.
2. GPTs를 어떻게 이해하면 좋은가
구분 일반 ChatGPT GPTs
사용 방식 대화마다 역할과 형식을 다시 설명 자주 쓰는 역할과 지시문을 미리 저장해 재사용
잘 맞는 일 빠른 질의응답, 초안 작성, 아이디어 정리 반복 리서치, 프롬프트 생성, 문서 템플릿 작업
오늘 수업 연결 심층 리서치 결과를 직접 확인 리서치 프롬프트 만들기, 구조화된 PPT 출력 흐름 고정
한 줄 정리
GPTs는 ChatGPT 안에 자주 쓰는 역할과 출력 형식을 저장해 두는 작업 전용 비서입니다. Gemini의 Gems와 같은 맥락으로 이해하면 됩니다.
3. 이번에 소개하는 GPTs
GPT 무엇을 해주나 언제 쓰나
심층 리서치 프롬프트 생성기 조사 주제와 독자, 목적을 받아 Deep Research용 질문을 설계 리서치를 시작하기 전에 조사 프롬프트를 잘 만들고 싶을 때
구조화된 프롬프트 생성기 원하는 입력과 출력 형식을 반영해 구조화된 프롬프트로 재정리 리서치 결과를 문서, 보고서, 슬라이드 형식으로 바꾸고 싶을 때
문서 템플릿 추출기 PPTX나 문서를 보고 JSON 구조의 템플릿으로 변환 기존 산출물 형식을 재사용 가능한 템플릿으로 뽑고 싶을 때
심층 리서치 프롬프트 생성기 열기 구조화된 프롬프트 생성기 열기 문서 템플릿 추출기 열기
4. 단계별 실습: 리서치 결과를 PPT로 바꾸기

여기서는 AI 인프라 수혜주 심층 리서치를 예시로, 템플릿 추출부터 최종 PPT 초안 생성까지 연결해 봅니다.

예시 PPTX 다운로드
단계 무엇을 하나 사용 도구
Step 1 예시 PPTX를 내려받아 기존 슬라이드 구조를 확인합니다. 예시 PPTX 파일
Step 2 PPTX를 업로드해 슬라이드 구조를 JSON 템플릿으로 추출합니다. 문서 템플릿 추출기
Step 3 AI 인프라 수혜주에 대한 Deep Research용 질문을 생성합니다. 심층 리서치 프롬프트 생성기
Step 4 생성된 질문으로 Deep Research를 수행해 핵심 결과를 확보합니다. ChatGPT Deep Research
Step 5 JSON 템플릿과 리서치 결과를 합쳐 PPT 출력용 구조화 프롬프트를 만듭니다. 구조화된 프롬프트 생성기
Step 6 최종 프롬프트를 ChatGPT에 넣어 슬라이드별 텍스트 초안을 생성합니다. ChatGPT
5. 바로 복사해 쓰는 요청 예시
심층 리서치 프롬프트 생성기 요청
기술에 익숙한 임원진을 위한 심층 리서치 프롬프트를 만들어 주세요. 주제: AI 인프라 수혜주 비교 대상 기업: NVIDIA, Broadcom, TSMC, SK hynix, Vertiv 목적: 향후 12~24개월 관점에서 어떤 기업이 가장 구조적 수혜를 받을지 비교 원하는 산출물: 1페이지 투자 메모 + 기업별 비교 표 + 리스크 3개 + 추가 확인 질문 3개
구조화된 프롬프트 생성기 요청
문서 템플릿 추출기로 얻은 JSON 템플릿과 심층 리서치 결과를 입력하면, 이를 PPT 초안으로 출력하는 구조화 프롬프트를 만들어 주세요. 목적: - 임원 보고용 PPT 텍스트 초안 생성 - 각 슬라이드는 제목, 핵심 메시지, 데이터 포인트, 시사점으로 구성 - 장표 수는 산출물당 1장씩.
최종 PPT 생성 요청
아래 두 자료를 바탕으로 임원 보고용 PPT 초안을 작성해 주세요. 자료 1: 문서 템플릿 추출기로 뽑은 JSON 템플릿 자료 2: AI 인프라 수혜주 심층 리서치 결과 작성 조건: - 슬라이드 구조는 JSON 템플릿을 참고한다. - 각 슬라이드는 제목, 핵심 메시지, 근거 데이터, 시사점을 포함한다. - 불필요한 수사는 줄이고, 투자위원회 메모 톤으로 쓴다. - 마지막 장에는 권고안과 추가 확인 질문을 넣는다.
