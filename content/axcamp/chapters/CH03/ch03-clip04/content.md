# ChatGPT 및 GPTs 소개

ChatGPT는 일반 채팅만 하는 도구가 아니라, **Deep Research**와 **GPTs**를 통해 반복 업무를 구조화할 수 있는 작업 플랫폼입니다.
이번 클립에서는 GPTs를 짧게 소개하고, **AI 인프라 수혜주 심층 리서치 결과를 PPT로 정리하는 실습 흐름**을 단계별로 확인합니다.

## 1. ChatGPT에서 먼저 볼 것

- [chatgpt.com](https://chatgpt.com/)에 접속해 로그인합니다.

- 일반 채팅, Deep Research, GPTs처럼 **작업 유형이 어떻게 나뉘는지** 먼저 확인합니다.

- 오늘은 단순한 질문보다, **반복 가능한 조사와 문서화 흐름**을 만드는 관점으로 ChatGPT를 봅니다.

### 한 줄 정리

GPTs는 ChatGPT 안에 자주 쓰는 역할과 출력 형식을 저장해 두는 **작업 전용 비서**입니다. Gemini의 Gems와 같은 맥락으로 이해하면 됩니다.

## 5. 바로 복사해 쓰는 요청 예시

심층 리서치 프롬프트 생성기 요청복사

기술에 익숙한 임원진을 위한 심층 리서치 프롬프트를 만들어 주세요.

주제: AI 인프라 수혜주 비교
대상 기업: NVIDIA, Broadcom, TSMC, SK hynix, Vertiv
목적: 향후 12~24개월 관점에서 어떤 기업이 가장 구조적 수혜를 받을지 비교
원하는 산출물: 1페이지 투자 메모 + 기업별 비교 표 + 리스크 3개 + 추가 확인 질문 3개

구조화된 프롬프트 생성기 요청복사

문서 템플릿 추출기로 얻은 JSON 템플릿과 심층 리서치 결과를 입력하면, 이를 PPT 초안으로 출력하는 구조화 프롬프트를 만들어 주세요.

목적:
- 임원 보고용 PPT 텍스트 초안 생성
- 각 슬라이드는 제목, 핵심 메시지, 데이터 포인트, 시사점으로 구성
- 장표 수는 산출물당 1장씩.

최종 PPT 생성 요청복사

아래 두 자료를 바탕으로 임원 보고용 PPT 초안을 작성해 주세요.

자료 1: 문서 템플릿 추출기로 뽑은 JSON 템플릿
자료 2: AI 인프라 수혜주 심층 리서치 결과

작성 조건:
- 슬라이드 구조는 JSON 템플릿을 참고한다.
- 각 슬라이드는 제목, 핵심 메시지, 근거 데이터, 시사점을 포함한다.
- 불필요한 수사는 줄이고, 투자위원회 메모 톤으로 쓴다.
- 마지막 장에는 권고안과 추가 확인 질문을 넣는다.
