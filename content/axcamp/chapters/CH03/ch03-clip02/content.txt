프롬프팅 기초

처음부터 무거운 업무 프롬프트로 들어가기보다, Gemini와 자연스럽게 대화하며 페르소나 설정, 맥락 누적, 구조화된 정리를 몸으로 익히는 클립입니다.

이번 실습의 흐름

1. Self Intro

자기소개 템플릿으로 대화를 시작하며 Gemini에게 대화 목적과 역할을 먼저 알려줍니다.

2. Follow-up

Gemini가 추가 질문을 하게 만들어 대화를 풍성하게 하고, 내 프로필과 관심사를 자연스럽게 구체화합니다.

3. Structured Output

대화 내용을 표, 불릿, Markdown 코드블록으로 정리하게 하면서 AI가 좋아하는 입력 형식을 체험합니다.

4. Canvas

정리된 프로필을 바탕으로 간단한 자기소개 웹페이지까지 요청해 보며 Gemini의 확장 기능을 가볍게 맛봅니다.

오늘 이 클립에서 익히는 것

좋은 프롬프트는 처음부터 복잡한 업무 지시문을 쓰는 것이 아닙니다. 대화의 목적을 먼저 정하고, 필요한 맥락을 쌓고, 결과를 구조화하는 습관이 출발점입니다.

1. 먼저 이렇게 시작해보세요

아래 프롬프트는 자기소개를 매개로 Gemini와의 첫 대화를 여는 템플릿입니다. 단순히 정보만 던지는 것이 아니라, Gemini가 인터뷰어처럼 질문을 이어가게 만드는 것이 핵심입니다.

자기소개 대화 시작 템플릿복사

당신은 AX Camp를 함께하는 AI 인터뷰어이자 프로필 정리 비서입니다.

지금부터 나와 대화하면서 내 업무 프로필을 함께 완성해 주세요.

규칙:
- 한 번에 질문은 1~2개만 해주세요.
- 내가 말한 내용은 기억하고, 다음 질문에 반영해 주세요.
- 답변 말미에는 "현재까지 이해한 나의 프로필"을 3줄 이내로 갱신해 주세요.
- 빠진 정보가 있으면 억지로 추정하지 말고 질문으로 채워 주세요.

먼저 아래 자기소개를 바탕으로 대화를 시작해 주세요.

- 이름:
- 소속/역할:
- 주요 경력:
- 관심 분야:
- 이번 교육에서 얻고 싶은 것:
- AI로 풀고 싶은 업무 문제:

2. Gemini에게 추가 질문을 부탁해보세요

한 번 답하고 끝내지 말고, Gemini가 나를 더 잘 이해하도록 질문을 이어가게 만드는 것이 중요합니다. 아래 요청을 한두 번 넣으면 프로필이 훨씬 입체적으로 잡힙니다.

추가 질문 요청복사

지금까지의 대화를 바탕으로, 내 프로필을 더 잘 이해하기 위해 필요한 질문을 5개만 골라 순서대로 해줘.

조건:
- 한 번에 1~2개씩만 질문
- 왜 이 질문이 필요한지도 한 줄로 덧붙이기
- 내 답변을 반영해서 다음 질문을 조정하기

프로필 보강 요청복사

지금까지 내용을 기준으로 내 프로필의 빈칸을 찾아줘.

아래 관점으로 아직 부족한 정보를 짚어주고, 필요한 것만 추가 질문해줘.
- 현재 맡고 있는 일
- 잘 아는 산업/도메인
- AI를 통해 바꾸고 싶은 업무
- 이번 교육에서 실제로 만들어보고 싶은 결과물

3. 대화 내용을 구조화해 보세요

여기서부터 구조화된 프롬프트의 가치가 드러납니다. 대화로 쌓은 내용을 일정한 형식으로 정리하게 만들면, 나중에 다시 쓰거나 다른 도구에 넘기기 쉬워집니다.

프로필 요약 요청복사

지금까지 대화를 바탕으로 내 프로필을 아래 형식으로 정리해줘.

## 기본 정보
## 핵심 경력
## 관심 산업 / 업무 영역
## 이번 교육에서 실험해보고 싶은 것
## AI와 함께 만들고 싶은 결과물

마지막에는 내가 Gemini에게 앞으로 어떤 질문을 하면 좋을지 3개 제안해줘.

Markdown 코드블록으로 저장하기복사

방금 정리한 프로필을 다시 한 번 정돈해서 출력해줘.

조건:
- 전체 내용을 ```markdown 코드블록``` 안에 넣어줘.
- 제목과 불릿을 유지해줘.
- 나중에 다른 AI 도구나 문서에 바로 붙여넣을 수 있게 깔끔하게 정리해줘.

왜 이 형식이 중요한가

AI는 긴 자유서술도 이해하지만, 제목·불릿·코드블록처럼 경계가 분명한 형식을 훨씬 안정적으로 다룹니다. 그래서 Markdown으로 정리된 결과는 저장하기도 좋고, 다음 프롬프트의 입력으로 다시 넣기도 좋습니다.

4. 한 걸음 더: Canvas로 확장해보기

정리된 프로필이 생기면, 이제 그것을 바탕으로 바로 무언가를 만들 수 있습니다. 여기서는 예시로 자기소개 웹페이지를 요청해보며 Canvas 계열 기능을 체감합니다.

프로필 기반 웹페이지 생성 요청복사

방금 정리한 내 프로필을 바탕으로, 한 페이지짜리 자기소개 웹페이지 초안을 만들어줘.

조건:
- 섹션: 소개 / 주요 경력 / 관심 분야 / AI로 만들고 싶은 것
- 톤: 깔끔하고 전문적인 스타일
- 너무 화려하지 않게
- 먼저 전체 구성안을 보여주고, 가능하면 Canvas에서 바로 수정할 수 있는 형태로 제안해줘.
