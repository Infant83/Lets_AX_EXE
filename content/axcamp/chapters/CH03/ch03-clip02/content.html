회의 오디오로 실습하는 4단계
회의 녹음과 맥락 자료는 모두 입력창 왼쪽의 파일 업로드 메뉴에서 붙인다. 핵심은 같은 대화창을 유지하는 것입니다.
단계입력AI에게 시키는 일산출물
Step 1회의 오디오구조화된 회의록 작성참석자, 발언 요지, 의사결정, 미해결 이슈
Step 2방금 만든 회의록전략 이슈와 의사결정 갭 추출핵심 이슈 목록, 리스크, 다음 회의 안건
Step 3회의 맥락 참고자료회의 내용과 데이터 교차검증불일치 포인트, 놓친 데이터, 재검토 항목
Step 4앞선 모든 분석 결과CEO용 의사결정 브리프 작성핵심 요약, 옵션 비교, 추천 액션

버튼을 누르면 실습용 구글 드라이브 링크가 클립보드에 복사됩니다.

프롬프트 Steps

아래 프롬프트를 그대로 붙여 넣은 뒤, 역할, 맥락, 출력 형식에 기반한 프롬프팅 기법의 중요성을 학습한다.

1. 음성 기반 AI 회의록 작성
첨부한 음성 파일은 LG그룹 하반기 AX(AI 전환) 전략 회의 녹음입니다. 이 음성을 듣고 아래 형식으로 구조화된 회의록을 작성해 주세요. ## 출력 형식 ### 1. 회의 기본 정보 - 회의명, 일시, 참석자(직책 포함) ### 2. 안건별 논의 내용 각 안건마다: - **안건 제목** - **발언 요약**: 누가 무슨 말을 했는지 핵심만 - **쟁점**: 의견이 갈린 부분 ### 3. 결정사항 - 번호 매기기, 담당자 명시 ### 4. 후속조치(Action Items) | 번호 | 조치 내용 | 담당자 | 기한 | |------|----------|--------|------| ### 5. 미결 사항 - 추가 논의가 필요한 이슈
2. 회의록 기반 핵심 이슈 추출
방금 작성한 회의록을 기반으로, 경영 전략 애널리스트 관점에서 분석해 주세요. ## 요청 사항 ### 1. 핵심 전략 이슈 TOP 3 각 이슈마다: - **이슈명**: 한 줄 요약 - **배경**: 왜 이것이 중요한지 - **회의에서의 논의 수준**: 충분 / 부분적 / 미흡 - **리스크 수준**: 🔴 높음 / 🟡 중간 / 🟢 낮음 ### 2. 의사결정 갭 분석 - 결정은 됐지만 실행 방안이 불명확한 항목 - 논의는 됐지만 결정이 안 된 항목 - 아예 다뤄지지 않았지만 다뤄졌어야 할 항목 ### 3. CEO 어젠다 추천 - 다음 회의에서 반드시 다뤄야 할 안건 2~3개 - 각 안건의 예상 소요 시간과 필요 참석자
3. 참고자료 첨부 + 멀티턴 분석
방금 첨부한 문서는 이번 회의의 배경이 되는 브리핑 자료입니다. (경쟁사 동향, 시장 데이터, 내부 KPI 등) 이 참고자료와 앞서 정리한 회의록을 교차 분석해서 다음을 정리해 주세요. ## 요청 사항 ### 1. 회의 논의 vs 실제 데이터 대조표 | 회의에서 언급된 내용 | 참고자료의 실제 데이터 | 일치 여부 | 비고 | |---------------------|----------------------|----------|------| ### 2. 계획 대비 실적 갭 - 상반기 목표 대비 실제 달성률 - 경쟁사 대비 상대적 위치 - 가장 큰 괴리가 발생한 영역과 원인 분석 ### 3. 회의에서 놓친 포인트 - 참고자료에는 있지만 회의에서 전혀 논의되지 않은 중요 데이터 - 해당 데이터가 의사결정에 미칠 수 있는 영향 ### 4. 데이터 기반 수정 권고 - 회의 결정사항 중 참고자료 데이터에 비춰 재검토가 필요한 항목 - 수정 방향 제안
4. 전략 추론 - CEO 의사결정 프레임워크
앞선 분석에서 가장 중요한 전략 이슈로 도출된 주제에 대해 딥 리서치를 실행해 주세요. ## 리서치 주제 "글로벌 AI 가전 시장에서 LG전자의 포지셔닝 전략: 삼성·중국 업체 대비 경쟁 우위 확보 방안" ## 리서치에 포함할 내용 1. 글로벌 AI 가전 시장 규모 및 성장 전망 (2025-2030) 2. 주요 경쟁사(삼성, 하이센스, 샤오미 등) AI 전략 비교 3. LG의 현재 AI 역량(EXAONE, ThinQ AI)의 시장 평가 4. 소비자 관점에서 AI 가전의 실질적 가치 5. LG가 집중해야 할 차별화 영역 제안 ## 출력 요건 - 각 항목에 출처 명시 - 데이터는 가능한 최신 자료 기반 - 최종 섹션에 'So What: LG 경영진을 위한 시사점' 포함
5. 딥 리서치 연계(추가)
지금까지의 모든 분석(회의록, 핵심이슈, 참고자료 교차분석)을 종합하여, CEO의 의사결정을 돕는 전략 보좌관 역할로 최종 권고안을 작성해 주세요. ## 출력 형식 ### 1. 상황 종합 (Executive Summary) - 3문장 이내로 현재 상황의 핵심을 요약 ### 2. 전략 옵션 비교표 각 주요 의사결정 사안에 대해: | 구분 | 옵션 A (공격적) | 옵션 B (점진적) | 옵션 C (보수적) | |------|----------------|----------------|----------------| | 핵심 내용 | | | | | 예상 투자 규모 | | | | | 기대 효과 | | | | | 주요 리스크 | | | | | 실현 가능성 | 상/중/하 | 상/중/하 | 상/중/하 | | 경쟁사 대비 포지션 | | | | ### 3. 추천안 및 근거 - 최우선 추천 옵션과 그 이유 (데이터 기반) - 차선책과 전환 트리거 조건 ### 4. 실행 로드맵 - 즉시 (1주 내): - 단기 (1개월 내): - 중기 (분기 내): ### 5. 모니터링 KPI - 핵심 지표 3~5개와 목표치
기억할 점
