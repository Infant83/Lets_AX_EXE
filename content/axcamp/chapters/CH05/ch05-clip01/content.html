Hi-D Code 소개 및 시연

최남석(Agentic AI 팀)이 보여주는 Hi-D Code의 핵심은 복잡한 작업을 한 번에 설명하고, 실행 결과를 빠르게 보는 데 있습니다. 오늘은 개념과 시연만 짧게 보고, 세부 자료는 CH07 참고자료 라이브러리에서 이어서 확인합니다.
무엇을 보나
  • 문제 정의를 짧게 설명하면, AI가 작업 단위를 나누는 흐름
  • 시연 결과를 바로 검토하고, 다시 손보는 반복 흐름
  • Agentic AI 팀이 실제로 보여주는 개발 자동화 감각
짧은 시연 흐름
단계 설명
1. 목표 입력 어떤 결과를 원하는지 짧게 설명합니다.
2. 실행 확인 AI가 작업을 어떻게 나누는지 보여줍니다.
3. 결과 검토 생성된 산출물을 보고 바로 수정 방향을 잡습니다.
다음 단계
오늘은 시연만 간단히 보고, 자세한 Agentic 자료와 도구 목록은 CH07 참고자료 라이브러리에서 이어서 봅니다.
